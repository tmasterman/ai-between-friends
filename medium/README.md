# Medium / Towards Data Science articles

This folder contains published Medium and Towards Data Science articles by Tula Masterman, stored in an LLM-friendly format.

## Suggested per-article format

For each article, create a folder with:

- `article.md` — the article text and metadata
- optional supporting assets

## Recommended headings

Use these headings to stay consistent and preserve provenance:

```md
# {Article Title}

- **Author:** Tula Masterman
- **Original publication:** Medium or Towards Data Science
- **Original link:** {URL}
- **Published:** {Date}
- **Imported to repo:** {Date}

## Summary

{Short summary}

## Article

{Full article text}
```
